Import RTF

Incarca un fisier RTF si creeaza automat variante, intrebari si raspunsuri.

Incarca test RTF

Format recomandat: VARIANTA, RAND, INTREBARE, PUNCTAJ, RASPUNS, CORECT.

Exemplu continut RTF: 
VARIANTA: Varianta 1
RAND: 1
INTREBARE: Ce este Node.js?
PUNCTAJ: 1
RASPUNS: Runtime JavaScript
RASPUNS: Baza de date
RASPUNS: Sistem de operare
CORECT: 1
